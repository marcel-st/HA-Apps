# Home Assistant Apps Store

XO Home Assistant private store - A custom add-on repository for Home Assistant.

## About

This repository contains custom Home Assistant add-ons maintained by Marcel Stangenberger.

**Repository URL:** https://github.com/marcel-st/hastore

## Available Add-ons

### SSH Call Home
Creates a reverse SSH tunnel to an external server, allowing remote access to your Home Assistant instance even when behind NAT or firewall.

**Version:** 1.0.6  
**Architectures:** aarch64, amd64, armv7

See [callhome/README.md](callhome/README.md) for detailed documentation.

## Installation

1. In Home Assistant, navigate to **Supervisor** > **Add-on Store**
